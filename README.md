# Draw the Shape — Precision Drawing Game

**Draw the Shape** is a fast-paced, skill-based game where your goal is simple:  **recreate geometric shapes as accurately as possible using your finger.**

It sounds easy — until you try to draw a perfect circle.

## 🧠 Core Idea

The game challenges your **precision, control, and spatial awareness**.

You are given a shape and must draw it freehand.  
Your drawing is analyzed and scored based on how close it is to the ideal form.

Every detail matters:
- symmetry  
- proportions  
- smoothness  
- closure  

## 🎯 Gameplay

- Choose a shape
- Draw it directly on the screen
- Get a **percentage score** based on accuracy
- Improve and beat your best result

As you progress:
- Shapes become more complex  
- Requirements become stricter  
- Your precision is pushed to the limit  

## 🧩 Game Features

- **15 unique shapes** across 5 levels
- Increasing difficulty system
- Real-time drawing with smooth curves
- Advanced scoring algorithm
- Personal best tracking
- Clean, modern UI with animated feedback

## 🏆 Level System

The game is structured into **5 levels**, each introducing new challenges:

| Level | Difficulty | Shapes Example |
|------|----------|----------------|
| 1 | Easy | Circle, Square, Triangle |
| 2 | Medium | Star, Diamond, Pentagon |
| 3 | Advanced | Hexagon, Arrow, Cross |
| 4 | Hard | Heart, Crescent, Rectangle |
| 5 | Expert | Octagon, Parallelogram, Spiral |

To advance:
- You must achieve a **minimum accuracy score**
- Complete all shapes in the level

## 📊 Scoring System

Your drawing is evaluated using multiple factors:

- **Shape accuracy** (how close to ideal geometry)
- **Line consistency**
- **Coverage** (did you complete the full shape?)
- **Closure** (did your line connect properly?)
- **Proportions**

The final score is a percentage reflecting overall precision.
